# CrossBlast Arduino Unity Rhythm Game

A rhythm game project that combines Arduino hardware input with a Unity-based interactive music game.

## Overview
This project was developed as a team-based game project combining physical input and digital gameplay.  
It connects Arduino button controls with a Unity rhythm game system to create a more immersive and interactive play experience.

The project includes both hardware input processing and in-game rhythm mechanics, covering the workflow from controller interaction to gameplay feedback.

## Main Features
- Arduino-based physical button input
- Unity rhythm game system
- Real-time note timing and key hit detection
- Score calculation and gameplay feedback
- Stage-based rhythm game flow
- Visual effects and UI interaction
- Sound synchronization with music
- Integrated hardware and software gameplay experience

## Tech Stack
- C++
- Arduino
- Unity
- C#

## Project Structure
This project includes:
- Arduino input handling code

- note spawning and timing system
- score and judgment system
- UI and visual feedback elements

- final game build and demo files
